Giai đoạn 1 — Khởi tạo dự án Java Web MVC

Bắt buộc

Mục tiêu: Tạo nền tảng ban đầu cho website, chạy được trên Tomcat và kết nối được MSSQL Server.

Công việc cần làm

Tạo project Java Web.
Cấu hình JDK 21.
Cấu hình Apache Tomcat.
Thêm MSSQL JDBC Driver.
Thêm JSTL nếu dùng JSP/JSTL.
Tạo cấu trúc package MVC.
Tạo file DBConnection.java.
Tạo file web.xml.
Tạo trang index.jsp.
Tạo layout header/footer cơ bản.

        ↓
Tạo THANH_TOAN với trạng thái Pending
        ↓
Giả lập callback thanh toán thành công/thất bại
        ↓
Nếu thành công:
    - Cập nhật THANH_TOAN = Thành công
    - Cập nhật DAT_CHO = Đã thanh toán
    - Cập nhật GIU_CHO = Đã chuyển vé
    - Tạo VE
    - Hiển thị mã vé
Nếu thất bại:
    - Cập nhật THANH_TOAN = Thất bại
    - Giữ chỗ vẫn còn đến khi hết hạn

6. Kết luận bản phân tích đúng

Xây dựng một trang web Hệ thống bán vé tàu gồm:

1. Website khách hàng công khai
   - Tìm chuyến
   - Chọn ghế
   - Đặt vé
   - Giữ chỗ
   - Thanh toán
   - Tra cứu vé
   - Trả vé online

2. Trang quản trị Admin nội bộ
   - Quản lý dữ liệu vận hành
   - Quản lý ga, tàu, toa, ghế
   - Quản lý chuyến tàu, lịch dừng
   - Quản lý bảng giá
   - Quản lý khuyến mãi
   - Quản lý ưu đãi
   - Quản lý chính sách đổi/trả
   - Quản lý đơn, vé, thanh toán, hoàn tiền
   - Báo cáo doanh thu

Không bao gồm:
- Quản lý nhân viên
- Bán vé tại quầy
- In vé giấy
Câu chốt nên dùng trong báo cáo: Hệ thống Web bán vé tàu được chia thành hai khu vực: khu vực khách hàng công khai phục vụ tra cứu, đặt vé, thanh toán và quản lý vé; khu vực Admin nội bộ phục vụ quản lý dữ liệu vận hành như ga, tàu, chuyến tàu, bảng giá, khuyến mãi, chính sách đổi/trả, đơn đặt chỗ, vé, thanh toán và hoàn tiền.
